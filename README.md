Food Ordering Web App

A modern and responsive web application for browsing, ordering, and managing food orders. Built with MERN Stack, this app allows users to view menus, add items to a cart, and place orders efficiently.


Features

- View categorized food menus (Starters, Mains, Desserts, etc.)
- Add/remove items from the cart
- View order summary with total price
- Checkout functionality (Payment Integration using Stripe)
- Filter or search food items
- Admin panel for managing menu (if applicable)
- Fully responsive UI

Tech Stack

- **Frontend:** HTML, CSS, JavaScript, React
- **State Management:** useState, useEffect, useContext
- **Styling:** Tailwind CSS 
- **Backend :** Node.js, Express
- **Database :** MongoDB
